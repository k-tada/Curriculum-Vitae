- 現状に満足することなく常に最新の技術をキャッチアップし、業務に活かすことを考えています。
  - 直近のプロジェクトに対して行った業務改善例

    - Jestの導入
  - それぞれの導入時において、提案のみならず導入までの調査や環境整備、技術ブログの執筆、必要なら社内にて勉強会の開催もすべて行っています。
